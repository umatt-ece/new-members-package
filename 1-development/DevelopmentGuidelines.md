# Development Guidelines

## Project Labels

```
 Department
  A: admin                      Project
  E: ece                           |
  M: mech        Human-readable    |
  O: other        Abbreviation     |
     |                 |           |
     E  000  -  X  -  XXXX  00  -  XX  00
         |      |           |          |
       GitHub   |        Revision     Year
       Issue    |        (version)
                |
           Document Type
            C: code
            D: drawing
```
